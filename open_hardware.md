[Mesh-CN](https://oshwhub.com/allrounderkali/fang-shui-he) is an open hardware carrier that mounts [ebyte modules](https://www.google.com/search?client=firefox-b-d&q=ebyte+sx1280) with A pro Micro

Pro micro options

| Board | Hardware Repo | Purchase | Availability |
|---|---|---|---|
| **nRFMicro** | [GitHub](https://github.com/joric/nrfmicro) | DIY only | Order PCBs from JLCPCB/PCBWay using provided gerbers |
| **BlueMicro** | [GitHub](https://github.com/jpconstantineau/NRF52-Board) | store offline | Pre-assembled available; stock varies |
| **Mikoto** | [GitHub](https://github.com/zhiayang/mikoto) | DIY only | No production files in latest revision; use v4.7 for JLCPCB |
| **IFNano** | [GitHub](https://github.com/joric/if-nano) | DIY only | 2-layer board, self-manufacture only |
| **Atlas Micro BLE** | [GitHub](https://github.com/ianchen06/atlas_micro_ble) | [Tindie](https://www.tindie.com/products/yhkeyboards/atlas-micro-ble/) | Listed on Tindie; check stock |
